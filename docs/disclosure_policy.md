# Disclosure Policy

## Principle

Release the methodology; protect the signal. The public repository is intended for methodological review, reproducibility, and academic use. 

## Public

- Method: public.
- Break-detection, segmentation, FFD, and d-invariance testing procedures: public.
- Public-safe notebooks, documentation, and configuration templates: public. 

## Private

- Signal: private.
- Raw data: excluded.
- Feature names: excluded.
- Feature decoder: excluded.
- Calibration schedules: excluded. 

## Allowed public artifacts

- Anonymised registry: allowed.
- The file `results/frozen_operators.csv` may be published only in anonymised, public-safe form.
- Public artifacts must not permit reconstruction of the proprietary feature universe, raw data, or operational calibration. 

## Rule

No file is committed until its disclosure status is classified as public, private, or publishable only after anonymisation.
